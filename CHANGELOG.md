# Changelog

## 0.1.0 - 2026-04-22

- Branded the repository and skill as `AgentTypeTest`.
- Renamed the main skill path to `skills/agent-type-test` and the core runner modules to `agent_type_test_*`.
- Moved saved development runs from `tmp/` into `examples/runs/` and kept `tmp/` as an ignored local scratch workspace.
- Added Orbit Core branding assets and GPT-5.4 example screenshots for the README.
- Added built-in local banks: `mbti93-cn` and `mini-ipip-en`.
- Added built-in website adapters: `dtti`, `16personalities`, and `sbti-bilibili`.
- Added deterministic reports in `json`, `md`, `html`, and `svg`.
- Added repository-level release metadata: GPL license, dependency list, and CI workflow.
